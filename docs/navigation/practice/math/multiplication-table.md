# 打印乘法口诀表

## Python 示例

以下是Python中打印乘法口诀表的示例代码：

```py
# title: 打印乘法口诀表
def print_multiplication_table():
    for i in range(1, 10):
        for j in range(1, i+1):
            print(f"{i}*{j}={i*j}", end="\t")
            
print_multiplication_table()

```
