# 查找元素是否在列表中

## Python 示例

以下是Python中查找元素是否在列表中的示例代码：

```py
# title: 查找元素是否在列表中
test_list = [ 1, 6, 3, 5, 3, 4 ]
print(4 in test_list)
print(test_list.count(4)>0)
print(5 in set(test_list))
```
