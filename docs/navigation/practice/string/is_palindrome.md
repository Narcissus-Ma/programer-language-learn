# 检查一个字符串是否是回文

## Python 示例

以下是Python中检查一个字符串是否是回文的示例代码：

```py
# title: 检查一个字符串是否是回文
def is_palindrome(string):
    return string == string[::-1]

def main():
    string = "racecar"
    print(is_palindrome(string))

if __name__ == '__main__':
    main()
```
