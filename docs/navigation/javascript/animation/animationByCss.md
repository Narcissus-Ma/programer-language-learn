```html

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />

    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

      #symbolExg {
        width: 50px;

          left: 0px;

          left: 100%;

        100% {
          left: 0px;
        }
      }
    </style>
```
